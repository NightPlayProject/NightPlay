# NightPlay

NightPlay is a lightweight Windows capture application for instant replay,
recording, screenshots, clip organization, editing, and sharing.

This repository is the official binary-release channel. NightPlay source code is
not published here. Download NightPlay only from the Releases page and verify the
published SHA-256 files before running it.

## NightPlay 0.0.3 beta

NightPlay 0.0.3 fixes beta-release detection in the setup program and completes
verified in-app update installs before restarting. It also retains responsive
large-library loading, bounded thumbnail memory, and capture customization.

NightPlay uses FFmpeg shared libraries under LGPLv2.1-or-later. Corresponding
source and build materials are attached to each applicable release.
